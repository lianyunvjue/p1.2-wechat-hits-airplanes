# p1.2-wechat-hits-airplanes
a little  game

wechat hits airplanes:

代码层面分析：
   
 角色：我方飞机 敌方飞机阵营 子弹    

功能：

    1.我方飞机:移动事件,发射子弹, 击中敌机,得分判断, 越界判断,暂停事件,结束判断
    
    2.敌方飞机:移动事件, 死亡判断, 越界判断,结束判断, 游戏难度控制
   
    3.子弹:发射行为,与敌机碰撞判断,越界判断 

使用技术：面向对象 定时器  添加事件  HTML CSS JS

游戏逻辑：

  1、创建角色      
  2、角色移动
  3.  敌方死亡判断
  4.得分判断       
  5、判断游戏是否结束          
  6、暂停页面,重新开始
